# Aplicação em Node 🚀

Api-movies é uma aplicação em Node.js onde o usuário cadastra um filme, preenche com algumas informações (nome, descrição, nota) e cria tags relacionadas a ele.

    Backend: Nodejs, Express
    Database: Sqlite, Knex
